| API response → React state | API data rendered directly in DOM — XSS risk if not escaped |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-12 | Tampering | search Input → brand/color filter params | mitigate | React JSX renders text as text nodes (not HTML) — no innerHTML usage; server validates via ilike() with parameterized SQLAlchemy, not raw concat |
| T-02-13 | Information Disclosure | FilamentTypeCard/Row renders API data | accept | All data is tenant-scoped at API layer; client-side rendering of tenant's own data is expected |
| T-02-14 | Tampering | has_sample toggle | mitigate | stopPropagation prevents double-trigger; mutation is idempotent (same value = no net change); server validates filament_type belongs to tenant |
